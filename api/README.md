# Automação de APIs

---
Bem-vindo à seção de Automação de APIs do nosso repositório! 
Esta área é dedicada ao uso e desenvolvimento voltado para a 
automação de testes de APIs, foco em proporcionar uma abordagem 
objetiva e eficiente para garantir que consiga utilziar e aplicar 
esta automação de forma rapida em seu projeto.

## Objetivo

Meu objetivo é oferecer ao QA um conjunto abrangente de 
ferramentas com maior presença no mercado e arquiteturas que 
facilitam a automação dos testes de APIs. Aqui, você encontrará 
exemplos práticos e arquiteturas detalhadas que demonstram 
como implementar soluções eficazes utilizando diferentes 
tecnologias.

## Tecnologias Utilizadas

Estamos comprometidos em oferecer suporte às principais tecnologias de automação de testes de APIs. Atualmente, nossas arquiteturas incluem:

1. **REST Assured (Java)**
    - Framework poderoso para testes de APIs RESTful em Java.
    - Facilita a validação de respostas de API e a verificação de endpoints.

2. **Robot Framework (JavaScript)**
    - Framework de automação de testes de código aberto que suporta testes em várias tecnologias.
    - Extensível com bibliotecas que permitem testes de APIs em JavaScript, oferecendo uma solução flexível e escalável.

3. **Postman**
   - Ferramenta popular para desenvolvimento e testes de APIs.
   - Oferece uma interface amigável e funcionalidades avançadas como testes automatizados, monitoramento e documentação.
